## Solid 演练场

访问[演练场](https://playground.solidjs.com/)，在浏览器中体验 Solid。

此外，我们还提供了 [JavaScript](https://stackblitz.com/github/solidjs/templates/tree/master/js) 和 [Typescript](https://stackblitz.com/github/solidjs/templates/tree/master/ts) Stackblitz 入门程序，它提供了一个基于 Web 的开发环境帮助您入门。

## 创建一个 Solid 应用

<Callout title="先决条件">

    - 熟悉命令行
    - 安装 [Node.js](https://nodejs.org/en)

</Callout>

Solid 提供了快捷的项目模板，帮助你启动。要运行应用程序，请选择您喜欢的语言执行一下步骤：

### JavaScript 项目

1. 在终端中运行以下命令获取 JavaScript 入门模板：

<TabsCodeBlocks>
<div id="npm">
```bash frame="none"
npx degit solidjs/templates/js my-app
```
</div>

<div id="yarn">
	```bash frame="none" yarn dlx degit solidjs/templates/js my-app ```
</div>

<div id="pnpm">
	```bash frame="none" pnpm dlx degit solidjs/templates/js my-app ```
</div>

<div id="bun">
```bash frame="none"
bunx degit solidjs/templates/js my-app
```
</div>
</TabsCodeBlocks>

2. 导航至应用程序目录：

```bash frame="none"
cd my-app
```

3. 安装必要的依赖项：

<TabsCodeBlocks>
<div id="npm">
```bash frame="none"
npm install
```
</div>

<div id="yarn">```bash frame="none" yarn install ```</div>

<div id="pnpm">```bash frame="none" pnpm install ```</div>

<div id="bun">
```bash frame="none"
bun install
```
</div>
</TabsCodeBlocks>

4. 运行应用程序：

<TabsCodeBlocks>
<div id="npm">
```bash frame="none"
npm run dev
```
</div>

<div id="yarn">```bash frame="none" yarn dev ```</div>

<div id="pnpm">```bash frame="none" pnpm dev ```</div>

<div id="bun">
```bash frame="none"
bun dev
```
</div>
</TabsCodeBlocks>

这将启动开发服务，现在，打开浏览器并导航至 `localhost:3000` 查看应用程序。
